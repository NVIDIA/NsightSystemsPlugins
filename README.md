# Nsight Systems Plugins

[NVIDIA Nsight Systems](https://developer.nvidia.com/nsight-systems) is a multi-node, system-wide performance analysis tool.

This GitHub repository enables publishing third-party Nsight Systems plugins.
Plugins registered in this site appear in a list, pointed by the [Nsight Systems user guide](https://docs.nvidia.com/nsight-systems/UserGuide/index.html#nsight-systems-plugins), exposing the registered plugins to Nsight Systems users.

This GitHub repository does not host the plugins' code, binary code or installers. It provided information about the third-part plugins and where to find them.

- [View third-party plugins list](https://potential-adventure-2egyr1z.pages.github.io/)
- [Add a third-party plugin to the list](./ADD_PLUGIN.md)
